This is a small, lightweight infinite Pinboard application.

My motivation stemmed from a personal challenge to minimize reliance on too many external libraries, but also because I wanted to challenge myself too see if I could even create them myself.

The app is fully keyboard accessible, including arrow key navigation where needed (toolbars).

All data is currently stored client-side and persisted in local storage, with the help of **zustand**.

For a demonstration, please see the live demo above and if you're curious you can also view the source code.

<p className="text-center text-xs text-theme-500">note: dragging on mobile is not supported</p>
